CherryLink Related Block Addon - Дополнение для CherryLink, добавляющее настраиваемые блоки с ссылками для перелинковки статей между собой.

== Description ==

Плагин CherryLink Related Block — это дополнение для основного плагина CherryLink. Оно добавляет возможность вывода блока с ссылками в теле статьи или произвольном места шаблона. Ссылки подбираются похожие на статью, используя алгоритмы релевантности основного плагина.

Присутсвуют настройки шаблона вывода, возможность задать ссылки индивидуально для каждой статьи, не показывать ссылки, которые уже есть в самой статье и другие опции.

Здесь оставлю важные ссылки:

1. Страница плагина [CherryLink Related Block](https://seocherry.ru/dev/cherrylink-related-block/ "Тут можно узнать подробности")
2. Ссылка на основной плагин [CherryLink](https://seocherry.ru/dev/cherrylink/ "Необходим для работы дополнения")

Чат в телеграме [Техподдержка](https://t.me/seovishnya "Обсуждение работы плагинов")

== Installation ==

1. Залейте файлы плагина в  `/wp-content/plugins/cherrylink` на вашем хостинге, или установить прямо из админки WP.
2. Активируйте на странице плагинов в админке WP
3. Перейдите в `Настройки->CherryLink` на вкладку `Блок ссылок` для изменения опций плагина

== Frequently Asked Questions ==

= Будет ли работать плагин без основного (CherryLink)? =

Нет не будет, он использует код и алгоритмы основного плагина.
